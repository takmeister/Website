Figure 1: My Raspberry Pi desktop

I use Raspberry Pi to develop AI and GUI software for robots, mainly for Eurobot. I use Python 2.7 with IDLE on the Pi, running a VNC server which I activate with SSH when I need it. My main Pi machine is a Raspberry Pi 3 with a 16 GB Kingston microSD card for boot and storage. I'm also a proud owner of the rare and elusive Raspberry Pi Zero which I'm currently looking for a project to use in.

I run a Raspbian Lite distribution with the Openbox Desktop Environment sporting a Windows 10-like colour scheme. The desktop wallpaper of Suzuka Circuit is an instance of feh running from autostart, and the htop process monitor is running in the background in a transparent tilda window. For the main tasks I use the terminator console and thunar file manager as shown in the picture above (Figure 1). The only cable coming from the Pi is for power as I use SSH and tightvncserver for when I need a GUI and the onboard WiFi chip takes care of internet access. I'd love to get Sublime Text 2 running as a text editor but for now my plain-text editors are nano and gedit, the latter for running under GUI.
